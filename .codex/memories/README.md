# Project Memories

This folder stores repo-local task memories. These are durable project notes that future Codex runs should read before continuing related work.

## Structure

- `tasks/`: one memory file per phase task ID.
- `task-memory-template.md`: template for new task memory logs.

## File Naming

Use:

```text
tasks/PPP_TTT-short-slug.md
```

Example:

```text
tasks/001_004-phase-task-memory-workflow.md
```

## Required Fields

Every task memory must include:

- Task ID
- Phase
- Task
- Goal
- Implementation log
- Verification
- Goal achieved
- Next idea or follow-up
